Descrizione
Questo progetto è un'applicazione web sviluppata utilizzando Angular, un framework TypeScript open-source sviluppato da Google. L'applicazione è stata creata con lo scopo di migliorare le mie capacità in front-end.

Requisiti di Sistema
Per eseguire correttamente questa applicazione, assicurati di avere installato quanto segue sul tuo sistema:

Node.js
npm (Node Package Manager)

Per installare e avviare l'applicazione sul tuo sistema, segui i passaggi seguenti:

Clona il repository sul tuo computer:
git clone https://github.com/tuo-nome-utente/nome-progetto.git

Entra nella directory del progetto:
cd nome-progetto

Installa le dipendenze del progetto:
npm install

Avvia l'applicazione:
npm start
Apri il browser e vai all'indirizzo: http://localhost:4200
